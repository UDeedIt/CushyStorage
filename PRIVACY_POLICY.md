# Privacy Policy for CushyStorage Library Demo

Effective Date: July 22, 2026

This Privacy Policy describes how the CushyStorage Library Demo ("the App"), developed by UDeedIt Pro, handles data. The App is a professional demonstration utility for the CushyStorage Android Library, designed to showcase secure and reactive data persistence.

## 1. Zero Data Collection Policy
CushyStorage is built on a "Privacy-First" architecture. We do not collect, transmit, or store any user data on external servers. All operations are performed locally on the host device.

## 2. Local Data Persistence & Encryption
The App demonstrates the library's multi-layered storage capabilities. Any data entered into the application is managed as follows:
*   Standard Persistence: Non-sensitive demonstration data is stored using the Android SharedPreferences system.
*   Secure Persistence: Sensitive demonstration data is protected using industry-standard AES-GCM encryption. All cryptographic keys are managed by the hardware-backed Android KeyStore and are inaccessible to external processes or servers.

## 3. Third-Party Integration
This App is a pure utility. It does not contain third-party tracking, advertising, or analytics SDKs. No data is shared with third-party service providers.

## 4. Required Permissions
The App does not require access to sensitive device permissions (e.g., location, camera, or contacts) to provide its core functionality.

## 5. Data Deletion
Data persisted during the demo is scoped to the application. All stored values are permanently purged by the Android system upon uninstallation of the App.

## 6. Commitment to Developers
As a D2D (Developer-to-Developer) product, we are committed to transparency and security. The logic demonstrated in this app is identical to the logic provided in the official CushyStorage library distribution.
